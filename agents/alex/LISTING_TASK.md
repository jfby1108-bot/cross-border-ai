## 🎯 目标

---

## 📋 调研任务

### 1. 各平台 Listing 要求

| 国家/地区 | 平台 | 调研内容 |
|-----------|------|----------|
| **美国** | Amazon | 标题要求、关键词、要点、图片规格、A+ |
| **美国** | eBay | 标题、描述、图片、分类 |
| **美国** | Walmart | Listing 要求 |
| **欧洲** | Amazon DE/UK/FR/ES | 小语种要求 |
| **欧洲** | Allegro (波兰) | 热门平台 |
| **日本** | Amazon JP | 日语要求 |
| **日本** | Rakuten | 乐天 Listing 特点 |
| **澳洲** | Amazon AU | 本地要求 |
| **澳洲** | eBay AU | 本地要求 |
| **中国** | 速卖通 AliExpress | 跨境要求 |

### 2. 调研维度

| 维度 | 内容 |
|------|------|
| **标题规则** | 字符限制、关键词顺序、品牌格式 |
| **描述要求** | 长度、格式、必需元素 |
| **图片要求** | 尺寸、格式、数量、主图要求 |
| **关键词** | 搜索词、类目节点 |
| **合规** | 认证、标签、禁售品 |

### 3. 一键部署方案

**目标**：客户输入产品信息 → 自动生成各平台 Listing

**需要的技术**：
- 多平台标题模板
- 关键词库（各平台）
- AI 描述生成（多语言）
- 图片标准化处理
- 类目自动匹配
