# Amazon Listing 详细规范

**平台**：Amazon（美国/欧洲/日本）  
**目标**：一键生成符合要求的 Listing  
**创建**：2026-03-21

---

## 🇺🇸 Amazon.com（美国站）

### 标题规范

| 项目 | 要求 |
|------|------|
| **最大长度** | 200 字符 |
| **推荐长度** | 150-180 字符 |
| **格式** | `品牌名 + 核心关键词 + 主要特性/型号 + 规格/颜色/尺寸 + 数量` |

**示例**：
```
Sony WH-1000XM5 Wireless Noise Cancelling Headphones - 30 Hour Battery - Black
```
- 品牌：Sony
- 核心词：Wireless Noise Cancelling Headphones
- 特性：30 Hour Battery
- 颜色：Black

### Bullet Points 规范

| 项目 | 要求 |
|------|------|
| **数量** | 5 个 |
| **每条最大** | 500 字符 |
| **推荐长度** | 100-200 字符/条 |
| **内容** | 核心卖点 + 关键参数 |

**格式建议**：
1. 第一条：品牌/核心价值
2. 第二三条：主要功能
3. 第四条：规格参数
4. 第五条：使用场景/售后

### 图片规范

| 类型 | 要求 |
|------|------|
| **主图** | 白底、无水印、1000x1000px、正方形 |
| **辅图** | 最多 7 张，场景图/细节图/对比图/infographic |
| **视频** | 可选，增强版（A+） |

### 关键词策略

**搜索词位置**：
1. 标题（最重要）
2. Bullet Points
3. 搜索词（后台）
4. 描述

## 🇪🇺 Amazon 欧洲站

### 站点

| 站点 | 国家 | 语言 | 主要要求 |
|------|------|------|----------|
| amazon.de | 德国 | 德语 | VAT + EPR |
| amazon.co.uk | 英国 | 英语 | UK CA |
| amazon.fr | 法国 | 法语 | VAT + EPR |
| amazon.es | 西班牙 | 西班牙语 | VAT + EPR |
| amazon.it | 意大利 | 意大利语 | VAT + EPR |

### 合规要求

- **VAT**：必须注册 VAT 税号
- **EPR**：德国/法国需要包装法注册
- **CE 标志**：需要 CE 认证的产品

## 🔧 一键生成模板

### 标题变量

```
{品牌} {核心产品词} {主要特性1} {主要特性2} {颜色/规格} {数量}
```

### Bullet 模板

```
1. 【{品牌}】{核心卖点}
2. {功能亮点1}，{功能亮点2}
3. 规格：{关键参数}
4. {使用场景}
5. {售后服务}
```
