## 🛒 Amazon（全球最大）

### 美国站 (amazon.com)

| 元素 | 要求 |
|------|------|
| **标题** | 最多 200 字符，建议 150-180 字符<br>格式：品牌 + 核心词 + 特性 + 规格 + 包装数 |
| **Bullet Points** | 5 个，每个最多 500 字符 |
| **描述** | A+ 推荐（品牌故事 + 图片） |
| **图片** | 主图：白底，1000x1000px<br>辅图：7 张，支持场景图/ infographic |
| **关键词** | 搜索词 + 类目节点 |

### 欧洲站 (DE/UK/FR/ES)

| 元素 | 要求 |
|------|------|
| **语言** | 本地语言（德语/英语/法语/西班牙语） |
| **VAT** | 需要 VAT 税号 |
| **合规** | CE 标志、 EPR 包装法 |

### 日本站 (amazon.co.jp)

| 元素 | 要求 |
|------|------|
| **语言** | 日语 |
| **标题** | 日文汉字 + かなカナ，100 字符内 |
| **图片** | 日本消费者偏好细节图 |

## 🛒 eBay

### 全球通用

| 元素 | 要求 |
|------|------|
| **标题** | 80 字符内，建议 60-70 字符 |
| **描述** | HTML 支持，可含图表 |
| **图片** | 12 张免费，eBay 会自动调整尺寸 |
| **分类** | 必须选择正确类别 |
| ** Item Specifics** | 品牌/材质/颜色等必填 |

## 🔧 一键部署技术方案

### 需要开发的模块

| 模块 | 功能 | 优先级 |
|------|------|--------|
| **标题生成器** | 品牌+关键词+特性 → 多平台标题 | P0 |
| **Bullet 生成** | 自动生成 5 个要点 | P0 |
| **描述生成** | AI 生成多语言描述 | P0 |
| **图片处理** | 自动裁剪/压缩/白底 | P1 |
| **关键词库** | 各平台搜索热词 | P1 |
| **类目匹配** | 自动匹配平台类目 | P2 |

### 技术实现

1. **模板系统**：各平台标题模板 + 变量替换
2. **AI 生成**：Qwen/GPT 生成描述和 Bullet
3. **图片处理**：Pillow 库自动处理
4. **API 集成**：Amazon SP API / eBay API
