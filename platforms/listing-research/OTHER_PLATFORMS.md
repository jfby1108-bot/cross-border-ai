# eBay / Walmart / Rakuten 详细规范

**平台**：eBay, Walmart, Rakuten, Allegro, 速卖通  
**目标**：多平台 Listing 生成  
**创建**：2026-03-21

---

## 🛒 eBay

### Listing 要求

| 项目 | 要求 |
|------|------|
| **标题** | 80 字符内，建议 60-70 字符 |
| **副标题** | 可选，55 字符 |
| **描述** | HTML 支持，最少 150 字符 |
| **图片** | 12 张免费，自动调整尺寸 |
| **分类** | 必须选择正确类别 |
| **Item Specifics** | 品牌/材质/颜色等必填 |

### 标题格式

```
[品牌] [产品类型] [型号/规格] [特性] [颜色/尺寸]
```

**示例**：
```
Apple iPhone 14 Pro Max 256GB - Space Black - Unlocked
```

### 多属性产品（Variations）

- 颜色、尺寸等使用 Variation
- 每个属性单独定价

## 🛒 Walmart

### 要求

| 项目 | 要求 |
|------|------|
| **标题** | 75 字符内 |
| **描述** | 纯文本，最少 100 字符 |
| **图片** | 白色背景，1500x1500px |
| **价格** | 需要有竞争力，Walmart 比价 |
| **配送** | 2-day shipping 推荐 |

### 特点

- 需要 Walmart seller 账号审批
- 品类限制较多
- 侧重自有物流（WFS）

---

## 🛒 Rakuten（乐天）

### 日本市场特点

| 项目 | 要求 |
|------|------|
| **标题** | 50 字符内，关键词靠前 |
| **描述** | 乐天专用编辑器，支持图片/视频 |
| **图片** | 最多 10 张，高质量场景图 |
| **语言** | 日语 |

### 日本消费者偏好

- 重视产品包装设计
- 详细的产品说明
- 实物图 + 场景图
- 评价很重要（积累好评）

## 🛒 Allegro（波兰）

### 欧洲市场

| 项目 | 要求 |
|------|------|

| **标题** | 80 字符内 |
| **物流** | Allegro Fulfillment 可用 |

## 🛒 速卖通（AliExpress）

### 跨境特点

| 项目 | 要求 |

| **图片** | 主图白底，支持视频 |
| **物流** | 俄/巴/欧美市场为主 |
