# AI 产品描述生成 - 技术架构

## 系统概述

基于 Qwen API 的跨境电商产品描述自动生成系统

## 架构图

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────┐
│   输入层        │     │   处理层         │     │   输出层        │
│  - 产品信息     │────▶│  - Qwen API     │────▶│  - 多语言描述   │
│  - 目标市场     │     │  - 模板引擎      │     │  - SEO 优化     │
│  - 语言偏好     │     │  - 内容校验      │     │  - 变体描述     │
└─────────────────┘     └──────────────────┘     └─────────────────┘
```

## 技术栈

- **Python 3.10+**
- **API**: Qwen (通义千问)
- **框架**: FastAPI (可选)
- **配置**: YAML

## 核心模块

### 1. product_describer.py
产品描述生成核心模块

### 2. qwen_client.py
Qwen API 客户端封装

### 3. config.yaml
配置文件

### 4. examples/
示例数据

## 使用流程

1. 导入产品信息（名称、特点、卖点）
2. 配置目标市场（美国/欧洲/日本等）
3. 选择输出语言
4. 调用生成 API
5. 获取优化后的产品描述
