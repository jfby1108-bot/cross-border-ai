# SupportIQ - 跨境电商 AI 客服

## 🚀 快速开始

### 在线演示
- **邮件客服**: https://cross-border-ai.onrender.com
- **状态**: 运行中 ✅

### 本地运行

```bash
# 克隆项目
git clone https://github.com/jfby1108-bot/cross-border-ai.git
cd cross-border-ai/mvp/AI-Customer-Service

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入邮箱配置

# 启动服务
python email_service.py
```
