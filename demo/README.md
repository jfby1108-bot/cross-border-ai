# 🛒 跨境电商 AI 产品描述生成器 - Demo

## 项目概述

这是一个专为跨境电商卖家设计的 AI 产品描述生成器 Demo，帮助快速生成符合目标市场习惯的产品描述。

### 功能特性

- ✨ **智能生成** - 基于 AI 自动生成吸引人的产品描述
- 🌍 **多语言支持** - 支持英语、法语、德语、西班牙语、日语等
- 🎯 **SEO 优化** - 内置关键词优化，提升搜索排名
- 📱 **多平台适配** - 支持 Amazon、eBay、速卖通等主流平台

## 快速开始

### 本地预览

```bash
# 方式 1: 直接用浏览器打开
open index.html

# 方式 2: 使用 Python 本地服务器
cd /path/to/demo
python3 -m http.server 8080
# 然后访问 http://localhost:8080

# 方式 3: 使用 Node.js http-server
npx http-server -p 8080
```

### 部署到 Netlify

```bash
# 1. 安装 Netlify CLI
npm install -g netlify-cli

# 2. 登录
netlify login

# 3. 初始化项目
netlify init

# 4. 部署
netlify deploy --prod
```

或者直接拖拽文件夹到 Netlify Drop (https://app.netlify.com/drop)

## 演示脚本

### 演示流程（约 5 分钟）

#### 1. 开场白（30 秒）
> "大家好，我是 Zane，今天向大家演示我们的跨境电商 AI 产品描述生成器。这个工具可以帮助卖家快速生成符合目标市场习惯的产品描述，大幅提升运营效率。"

#### 2. 基础功能演示（2 分钟）

**演示步骤：**
1. 打开 Demo 页面
2. 输入产品名称："无线蓝牙降噪耳机"
3. 选择产品类别："电子产品"
4. 选择目标市场："美国 (Amazon)"
5. 输入产品特点："主动降噪、30小时续航、蓝牙5.0"
6. 点击"生成产品描述"

**讲解要点：**
> "大家可以看到，只需输入产品基本信息，AI 就会自动生成一段专业的产品描述。描述包含了产品特点、使用场景、客户评价等要素，完全符合 Amazon 的格式要求。"

#### 3. 多语言演示（1 分钟）

**演示步骤：**
1. 保持产品信息不变
2. 将目标市场切换到"德国 (Amazon DE)"
3. 再次点击生成

**讲解要点：**
> "同一款产品，切换到德国市场后，AI 自动生成了德语版本的产品描述。这大大节省了卖家翻译和本地化的时间。"

#### 4. 案例展示（1 分钟）

**演示步骤：**
1. 点击"案例展示"标签
2. 点击"天然有机护肤套装"案例
3. 展示生成结果

**讲解要点：**
> "我们还内置了多个行业的优秀案例，大家可以参考学习。每个案例都经过了专业优化，符合目标市场的表达习惯。"

#### 5. 结尾（30 秒）

> "这就是我们的 AI 产品描述生成器。目前是 Demo 版本，欢迎大家试用并提出宝贵意见。如有兴趣了解更多或合作，请联系我们的团队。谢谢大家！"
