# 客户演示脚本 - 15分钟

## 🎯 目标
让客户了解 SupportIQ AI 客服的价值，推动试用/付费

---

## 📝 演示流程 (15分钟)

### 1. 开场 (2分钟)
```
"Hi [客户名]，感谢时间。我是 SupportIQ 的 Casey。

我们专注跨境电商 AI 客服，帮您：
- 7×24 小时自动回复（夜间询盘不丢失）
- 30+ 语言自动切换
- Amazon/Shopify/eBay 多平台统一管理

今天大概 15 分钟，我先看看您目前的客服情况？"
```

### 2. 痛点了解 (3分钟)
```
"您现在客服团队几个人？主要哪些平台？"
"有没有遇到过夜间询盘错过的情况？"
"客服成本大概多少？"
```

### 3. 产品展示 (5分钟)
```
"我给您展示一下我们的系统..."
(演示邮件自动回复、多语言切换)
```

### 4. 案例分享 (2分钟)
```
"深圳一家电子公司，用了我们后：
- 客服响应速度提升 80%
- 夜间询盘不再丢失
- 每月省 $3000+ 人工成本"
```

### 5. 推动试用 (2分钟)
```
"您要不要免费试用 7 天？体验一下效果？"
```

### 6. 结束 (1分钟)
```
"谢谢您的时间！我会把演示链接发您微信。"
```

---

## 💡 关键话术

### 被拒绝时
"理解，那您有朋友感兴趣可以推荐一下，谢谢！"

### 有兴趣时
"好的，我帮您安排账号，下周一就能开始试用。"

### 问价格时
"我们有 3 个套餐：
- 基础版 $99/月（自动回复）
- 专业版 $199/月（多语言+多平台）
- 企业版 $499/月（定制+优先支持）"
